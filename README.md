# 💰 Finance Dashboard

A simple and interactive finance dashboard built using React.js to track income, expenses, and financial insights.

---

## 🚀 Live Demo

https://finance-dashboard-nb.netlify.app/

---

## 📌 Features

* Dashboard summary (Income, Expense, Balance)
* Transactions list with details
* Search and filter transactions
* Role-based UI (Viewer & Admin)
* Insights (spending analysis)
* Responsive design

---

## 🛠️ Tech Stack

* React.js
* Context API
* CSS
* Recharts

---

## ⚙️ How to Run

```bash
npm install
npm run dev
```

---

## 📊 Functionality

* Viewer → Can only view data
* Admin → Can add transactions
* Filters → Search and filter by type
* Insights → Shows income, expense, and status
